# Telegram Automation System

Система автоматизации работы Telegram-аккаунта (userbot) для контролируемой рассылки сообщений и автоматической обработки входящих сообщений.

## ⚠️ Предупреждение

Используйте систему на свой страх и риск. Нарушение правил Telegram может привести к блокировке аккаунта.

## 🚀 Возможности

- **📤 Рассылка сообщений** — отправка по списку чатов с использованием шаблонов
- **📷 Рассылка с фото** — поддержка медиа-вложений
- **🛡️ Анти-бан защита** — рандомизация задержек, лимиты, паузы
- **💬 Автоответы** — обработка входящих сообщений
- **📊 Логирование** — полная запись всех действий
- **🗄️ SQLite база** — оптимизированное хранение истории
- **⚙️ Гибкая конфигурация** — настройки через JSON
- **📋 Сбор чатов** — автоматическое получение списка чатов из аккаунта

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка API credentials

Откройте `.env` и заполните:

```env
API_ID=26680307
API_HASH=11fe6a7625e5c368e494c40f29066de0
PHONE=+79124922318
```

**Как получить API ключи:**
1. Перейдите на https://my.telegram.org
2. Авторизуйтесь по номеру телефона
3. Нажмите "API development tools"
4. Создайте новое приложение
5. Скопируйте `api_id` и `api_hash` в `.env`

### 3. Запуск главного меню

```bash
python start.py
```

**Меню предложит:**
1. 🔑 Войти по номеру (создать сессию + собрать чаты)
2. 📡 Запустить рассылку (используя сессию)
3. 📋 Проверить статус сессии
4. 📊 Показать статистику чатов
5. 🚪 Выход

### 4. Первый запуск (авторизация)

1. Выберите пункт **1. Войти по номеру**
2. Введите код из Telegram (придёт в приложение или SMS)
3. Если включена 2FA — введите пароль
4. Скрипт автоматически соберёт все чаты и сохранит в `config/chats.json`

### 5. Настройка рассылки

1. Откройте `config/templates.json` — отредактируйте шаблоны сообщений
2. Откройте `config/chats.json` — отключите ненужные чаты (`"enabled": false`)
3. Выберите пункт **2. Запустить рассылку**

## 🛡️ Защита от блокировки

Система автоматически применяет:
- **Дневной лимит**: 40 сообщений
- **Часовой лимит**: 10 сообщений
- **Задержка между сообщениями**: 90-180 секунд
- **Рандомизация**: перемешивание чатов

**Рекомендации:**
- Начинайте с малых лимитов (20-30 сообщений/день)
- Постепенно увеличивайте активность
- Не используйте на основном аккаунте без тестирования
