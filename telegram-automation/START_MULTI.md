# 🚀 ЗАПУСК МУЛЬТИАККАУНТНОЙ РАССЫЛКИ

## Быстрый старт за 5 шагов

### Шаг 1: Проверка базы чатов и шаблонов
```bash
# Убедитесь, что файлы существуют:
config/chats.json      ← База чатов (102 чата найдено ✓)
config/templates.json  ← Шаблоны (5 шаблонов ✓)
```

### Шаг 2: Добавление аккаунтов
```bash
python multi_account_start.py
```
В меню:
- Выберите **"3. Добавить аккаунт"** (повторите для каждого аккаунта)

### Шаг 3: Авторизация аккаунтов
```bash
# Вариант A: Через главное меню
python multi_account_start.py
→ "4. Проверить сессии" (для проверки)
→ "2. Аккаунты" → введите ID аккаунта → авторизовать

# Вариант B: Быстрая авторизация
python auth_accounts.py
```

### Шаг 4: Настройка скриптов и фото
```bash
python multi_account_start.py
→ "2. Аккаунты"
→ Введите ID аккаунта для редактирования
→ "6. Настроить скрипт" (выберите шаблон или введите текст)
→ "7. Настроить фото" (использовать default или свой путь)
```

### Шаг 5: Запуск рассылки
```bash
python multi_account_start.py
→ "1. Запустить рассылку"
→ Выберите стратегию распределения (1=равномерно, 2=пропорционально)
→ Используйте фото? (y/n)
→ Готово!
```

## 📊 Пример для 2 аккаунтов

**Дано:**
- 102 чата в базе
- 2 аккаунта готовы к работе

**Распределение:**
- Аккаунт 1: 51 чат
- Аккаунт 2: 51 чат

**Время рассылки:**
- Задержка: 50-100 секунд между сообщениями
- Итого: ~1.5-2 часа на цикл

---

## 📈 Масштабирование до 10-15 аккаунтов

**Дано:**
- 1000 чатов
- 10 аккаунтов

**Распределение (равномерно):**
- Каждый аккаунт: 100 чатов

**Время рассылки:**
- ~2-3 часа на цикл (с задержками 50-100 сек)

## ✅ Чеклист перед запуском

- [ ] `accounts.json` содержит аккаунты с `api_id`, `api_hash`, `phone`
- [ ] Все аккаунты авторизованы (сессии созданы)
- [ ] `config/chats.json` содержит чаты
- [ ] `config/templates.json` содержит шаблоны
- [ ] Для каждого аккаунта настроен скрипт (шаблон или текст)
- [ ] Для каждого аккаунта настроено фото (опционально)

---

## 🐛 Если что-то пошло не так

### "Нет готовых аккаунтов"
→ Проверьте, что в `accounts.json` заполнены `api_id`, `api_hash`, `phone`

### "Не авторизован"
→ Запустите `python auth_accounts.py` для авторизации

### "Чаты пусты"
→ Проверьте `config/chats.json`

### "FloodWait"
→ Увеличьте задержки в настройках аккаунта
