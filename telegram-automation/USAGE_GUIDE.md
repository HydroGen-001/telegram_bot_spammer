## 📥 ВСТУПЛЕНИЕ В ЧАТЫ

Для массового вступления в 3500+ чатов.

---

### Шаг 1: Распределение чатов

```bash
python scripts/distribute_chats.py
```

**Меню:**
```
1. Распределить чаты          → Создаст config/accounts/{id}/chats.json
2. Изменить кол-во чатов      → По умолчанию 200 чатов/аккаунт
3. Показать статистику        → Сколько чатов в базе
4. Открыть папку              → config/accounts/
0. Выход
```

**Что происходит:**
- Читает `config/chats.json` (3500 чатов)
- Распределяет по **200 чатов на аккаунт**
- Создаёт папки:
  - `config/accounts/1/chats.json`
  - `config/accounts/2/chats.json`
  - и т.д.

---

### Шаг 2: Вступление в чаты

```bash
python scripts/join_chats.py
```

**Меню:**
```
1. Вступить в чаты            → Запуск процесса
2. Настройки лимитов          → Изменить дневной лимит
3. Показать прогресс          → Статистика по аккаунтам
4. Сбросить прогресс          → Начать заново
5. Открыть папку              → config/accounts/
0. Выход
```

**Процесс вступления:**
1. Выберите аккаунт (1, 2, ...)
2. Скрипт вступает в чаты из `config/accounts/{id}/chats.json`
3. **Лимит:** 30 вступлений/день
4. **Задержка:** 45-90 сек между вступлениями
5. **Пауза:** 10-20 мин после каждых 5 вступлений

---

### Прогресс вступления

**Файл:** `config/accounts/{id}/progress.json`

```json
{
  "joined": [
    {"id": "chat1", "name": "Чат 1", "joined_at": "2026-03-23T10:30:00"}
  ],
  "failed": [
    {"id": "chat2", "name": "Чат 2", "error": "ChannelPrivate"}
  ],
  "last_join_date": "2026-03-23",
  "joins_today": 15
}
```

---

### Пример расчёта

**Дано:**
- 3500 чатов
- 200 чатов/аккаунт
- 30 вступлений/день

**Расчёт:**
```
3500 ÷ 200 = 18 аккаунтов нужно

Для 1 аккаунта (200 чатов):
  200 ÷ 30 = 7 дней

Итого: 7 дней на аккаунт × 18 аккаунтов = 126 дней
```

## ❓ РЕШЕНИЕ ПРОБЛЕМ

### ❌ FloodWait (блокировка от Telegram)

```
FloodWait: ждите 3600 сек
```

**Причина:** Слишком частые действия

**Решение:**
1. Увеличьте задержки в `start.py`:
   ```python
   MIN_DELAY = 90  # было 50
   MAX_DELAY = 180  # было 100
   ```
2. Уменьшите дневной лимит
3. Подождите указанное время

---

### ❌ Сессия не найдена

```
Сессия не активна
```

**Решение:**
```bash
python start.py → пункт 1
```
Пройдите авторизацию заново

---

### ❌ Шаблоны не найдены

```
Шаблоны не найдены: config/templates.json
```

**Решение:**
1. Проверьте путь в файле
2. Создайте `config/templates.json` по образцу

```

```

**Решение:**
- Нужна ссылка-приглашение (t.me/+abc123)
- Пропустите этот чат

### Встроенные лимиты

| Параметр | Значение |
|----------|----------|

1. **Начинайте с малых лимитов** (20-30 сообщений/день)
2. **Постепенно увеличивайте** активность
3. **Используйте прокси** (мобильные с ротацией)
4. **Не используйте основной аккаунт** для тестов
5. **Следите за логами** и ошибками
