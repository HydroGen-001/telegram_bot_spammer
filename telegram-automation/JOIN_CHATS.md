# ВСТУПЛЕНИЕ В ЧАТЫ

## Новый функционал

**Пункт 7 в главном меню: "Вступить в чаты (распределение)"**

---

## Как работает

### Алгоритм:

1. **Загрузка базы чатов** из `config/chats.json`
2. **Распределение** — по N чатов на каждый аккаунт
3. **Подключение** аккаунтов через прокси
4. **Проверка** — состоит ли аккаунт в чате
5. **Вступление** — если не состоит
6. **Статистика** — по каждому аккаунту и общая

---

## Использование

```bash
python multi_account_start.py
```

**Выберите пункт 7:**
```
  7. Вступить в чаты (распределение)
```

**Введите количество чатов на аккаунт:**
```
Сколько чатов выделять на аккаунт?
  Введите число (Enter=50): 50
```

## Настройки

### В файле `multi_account_start.py`:

```python
# Максимум вступлений за сессию
max_join_per_session=50

# Задержка между вступлениями (сек)
delay_between_joins=(10, 30)
```

### Изменить в меню:
- Введите своё число вместо 50

---

## Ограничения Telegram

| Лимит | Значение |
|-------|----------|
| Вступлений в день | ~50-100 |
| Вступлений в час | ~10-20 |
| Всего каналов | 500 |

**Рекомендация:** Не более 50 вступлений за сессию!

---

## Обработка ошибок

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `ChannelPrivate` | Частный канал | Нужно приглашение |
| `FloodWaitError` | Лимит Telegram | Ждать N секунд |
| `UserAlreadyParticipant` | Уже в чате | Пропускаем |
| `ValueError` | Чат не найден | Удалить из базы |

---

## Интеграция с рассылкой

После вступления в чаты:

1. **Запустите рассылку** (пункт 1)
2. Аккаунты будут рассылать по **своим чатам**
3. **Ошибок доступа не будет** — аккаунты уже в чатах

---

## Полный цикл работы

```
1. Импорт чатов из CSV (пункт 6)
   ↓
2. Вступить в чаты (пункт 7)
   ↓
3. Запустить рассылку (пункт 1)
   ↓
4. Получить статистику
```

## Пример для 3 аккаунтов

```
База: 3000 чатов
Аккаунты: 3

Распределение:
  Аккаунт 1: 50 чатов → Вступил
  Аккаунт 2: 50 чатов → Вступил
  Аккаунт 3: 50 чатов → Вступил

Остаток: 2850 чатов

На следующий день:
  Аккаунт 1: ещё 50 чатов → Вступил
  ...
```

**За 2 месяца:** 3 аккаунта × 50 чатов × 60 дней = **9000 чатов**
