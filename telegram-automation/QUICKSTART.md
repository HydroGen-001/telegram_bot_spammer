# 🚀 БЫСТРЫЙ СТАРТ

Telegram Automation — система для рассылки сообщений и автоответов.

---

## 📋 Шаг 1: Установка зависимостей

```bash
cd telegram-automation
pip install -r requirements.txt
```

---

## 🔑 Шаг 2: Получение API ключей

1. Перейдите на https://my.telegram.org
2. Авторизуйтесь по номеру телефона (в формате +7...)
3. Нажмите **"API development tools"**
4. Заполните форму:
   - **Title**: любое название (например, MyUserbot)
   - **Short name**: короткое имя (например, myuserbot)
   - **Platform**: Desktop
5. Нажмите **"Create application"**
6. Скопируйте:
   - **App api_id** (число)
   - **App api_hash** (строка)

## 📱 Шаг 4: Первый запуск (авторизация)

```bash
python start.py
```

**Выберите пункт 1 — Войти по номеру:**

1. Скрипт отправит код в Telegram
2. Введите код из приложения (или SMS)
3. Если включена 2FA — введите пароль
4. Скрипт автоматически соберёт все чаты

**Результат:**
- ✅ Создана сессия: `sessions/userbot.session`
- ✅ Собраны чаты: `config/chats.json`

## 📝 Шаг 5: Настройка шаблонов

Откройте `config/templates.json`:

```json
{
  "default_photo": "C:\\Users\\ACER\\OneDrive\\Рисунки\\photo.jpg",
  "templates": [
    {
      "id": 1,
      "name": "greeting",
      "text": "Добрый день! 👋\n\nХочу предложить сотрудничество...",
      "has_photo": false
    },
    {
      "id": 3,
      "name": "contract_offer",
      "text": "🫵 🫵 Сразу к делу...",
      "has_photo": true
    }
  ]
}
```

**Параметры:**
- `default_photo` — путь к фото для рассылок с медиа
- `templates` — список шаблонов сообщений
- `has_photo: true` — шаблон используется с фото

## 📋 Шаг 6: Настройка чатов

Откройте `config/chats.json`:

```json
{
  "chats": [
    {
      "id": "workarena",
      "name": "WORKARENA •ПОДРАБОТКИ МОСКВА",
      "type": "channel",
      "enabled": true
    },
    {
      "id": "disabled_chat",
      "name": "Неактивный чат",
      "type": "channel",
      "enabled": false
    }
  ]
}
```

**Как найти username чата:**
- Откройте чат в Telegram
- Посмотрите адрес вида `@username` (без @)
- Для частных чатов используйте ID

---

## 📤 Шаг 7: Запуск рассылки

```bash
python start.py
```

**Выберите пункт 2 — Запустить рассылку:**

Система покажет:
- 📝 Выбранный шаблон
- 📷 Путь к фото (если есть)
- ✅ Статус подключения

**В процессе рассылки:**
- ⏳ Обратный отсчёт до следующего сообщения
- ✅ Успешные отправки (с фото / только текст)
- ❌ Ошибки отправки

**Лимиты по умолчанию:**
- Дневной: 40 сообщений
- Часовой: 10 сообщений
- Задержка: 90-180 секунд

## ❓ Решение проблем

### ❌ "API credentials not found"
Проверьте `.env` — должны быть заполнены `API_ID`, `API_HASH`, `PHONE`

### ❌ "Сессия не найдена"
Запустите авторизацию: `python start.py → пункт 1`

### ❌ "FloodWait" (блокировка от Telegram)
- Увеличьте задержки в `start.py` (MIN_DELAY, MAX_DELAY)
- Уменьшите дневной лимит (DAILY_LIMIT)
- Подождите указанное время

### ❌ "Шаблоны не найдены"
Проверьте путь в `TEMPLATES_PATH` — должен указывать на `config/templates.json`

### ❌ "Чаты не найдены"
Запустите авторизацию — чаты собираются автоматически
